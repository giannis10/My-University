## My University

Μια εφαρμογή “φοιτητικού πορτοφολιού” που στοχεύει να κάνει τη φοιτητική ζωή **πιο οργανωμένη και παραγωγική**: πρόγραμμα, σημειώσεις, βαθμολογίες, κάρτες/έγγραφα, ρυθμίσεις προφίλ και συγχρονισμός.

Το project είναι **φτιαγμένο από φοιτητή Cyber Security για φοιτητές**. Αναπτύσσεται και χρησιμοποιείται **τα τελευταία 2+ χρόνια**, με συνεχείς βελτιώσεις και αναβαθμίσεις.

### Τι προσφέρει (ενδεικτικά)

- **Πρόγραμμα μαθημάτων & ημερολόγιο**
- **Σημειώσεις**
- **Βαθμολογίες / πρόοδος μαθημάτων**
- **Δρομολόγια / χρήσιμες πληροφορίες**
- **Κάρτες/έγγραφα** (σε μορφή “wallet”)
- **Προφίλ & ρυθμίσεις**
- **Συγχρονισμός δεδομένων** ανά χρήστη

### Roadmap / Επόμενες αναβαθμίσεις

- Βελτιώσεις εμπειρίας χρήσης (UX)
- Περισσότερες λειτουργίες οργάνωσης/παραγωγικότητας
- Καλύτερη offline εμπειρία όπου χρειάζεται
- Νέες δυνατότητες βάσει feedback φοιτητών
